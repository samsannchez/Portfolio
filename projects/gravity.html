3D Gravity Simulator

Overview

This project will emulate a real out-of-space enviroment and how gravity has an effect on planets in the solar system. We will utilize physics applied mathematics, C++ programming and OpenGL library. I will not be using any rendering softwares such as Unity or Unreal engine. I chose not to use these tools because I want to be able to do all the rendering.
