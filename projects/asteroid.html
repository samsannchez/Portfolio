Asteroids Game

For this project I programmed a classic asteroids game using 3 classes on Java. The first class is called Asteroids.class and its function is to generate the asteroids. The asteroids are placed on my GUI sandbox by generating a shape of random size and a random number of vertices. The next class is Rocket.class this class will define the shape of our rocket. For this specific project I have chosen a simple triangular shape. This shape is defined by providing the angles and will not change. Finally, we have asteroidsgame.class. Here is where all the components will come together and where the functionallity of the game is defined.
